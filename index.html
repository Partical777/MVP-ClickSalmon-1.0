Get start for tracking click on your web

Step. 1

Let’s take a look this is Click Salmon beta website. Its special point is generating the tracking route. Everyone could easily see which button or route is most people explored. It can help you improve UI/UX of website. Even find something unexpected route from users.


Step. 2

Ok, let’s refresh the page. Set your ID. (Every user can set any ID in beta version. So if you find something in tree diagram after setting ID, please change an ID. Thank you! We’ll improve it.)


Step. 3

Next, you can see your ID at the top-right corner. And the field of tree diagram should be empty. Oh, there’s only one dot.


Step. 4

Click the copy, copying the 3 lines cdn code.


Step. 5

Open an example web.


Step. 6

Use any editor, open all the html files you want to track. Paste above </body> (line404)


Step. 7

Ok, you’re done. That’s simple, right? Go back your example website, refresh and open the chrome developer console. You’ll see this line “Document written with ID : **********”. Then, click any button multiple times.


Step. 8

Open “Click Salmon” again. There’s a line there.


Step. 9

Ok. We’re done. Let’s click more. Open the web on the new tab. (Because if you keep clicking on the same tab, it will still track on the same line)


Step. 10

Finally, you will see a complete tree diagram.
